# 🔍 Аудит парсеров — ИТОГОВЫЙ ОТЧЁТ

## ❌ Текущее состояние: НИ ОДИН парсер не работает

### Причина: IP блокировка

Все крупные российские сайты недвижимости блокируют запросы с IP дата-центров:

| Сайт | HTTP (httpx) | Playwright | cianparser | Причина |
|------|-------------|------------|------------|---------|
| **ЦИАН** | ❌ 403 WAF | ❌ "Подозрительный трафик" | ❌ Зависает | IP дата-центра в чёрном списке |
| **Авито** | ❌ Пустая страница | ❌ "Доступ ограничен" | — | IP заблокирован |
| **Домклик** | ❌ 401 | ❌ 403 | — | IP не из России / дата-центр |

**Вывод:** Любой HTTP-клиент (httpx, requests, cloudscraper) и headless-браузер (Playwright, Selenium) с этого сервера будет заблокирован.

## ✅ РЕАЛЬНЫЕ решения

### Вариант 1: Резидентные прокси (рекомендуется)

```python
# Используем прокси с российскими residential IP
PROXIES = [
    "http://user:pass@proxy1.ru:8000",
    "http://user:pass@proxy2.ru:8000",
]

# Провайдеры прокси:
# - SmartProxy (smartproxy.com) — от $7/ГБ
# - Bright Data (brightdata.com) — от $5/ГБ  
# - IPRoyal (iproyal.com) — от $1.75/ГБ
# - Geonode (geonode.com) — от $2/ГБ
```

Стоимость: ~$50-100/месяц для парсинга 100K объявлений.

```

```
